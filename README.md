# 📰 Crypto News Aggregator

> A lightweight, open-source cryptocurrency news aggregator that pulls real-time news from multiple sources into one clean feed.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.8+-green.svg)
![Status](https://img.shields.io/badge/status-active-brightgreen.svg)

## ✨ Features
- 🔴 Real-time news from CoinDesk, CoinTelegraph, Decrypt, and more
- 🏷️ Filter by coin — search news by Bitcoin, Ethereum, Solana, etc.
- 📊 Sentiment analysis — positive/negative/neutral scoring
- 💾 Export to JSON/CSV
- 🖥️ CLI interface — simple and fast
- 🌐 Web dashboard — clean browser UI

## 🚀 Quick Start

### Installation
\`\`\`bash
git clone https://github.com/jaelani27/crypto-news-aggregator.git
cd crypto-news-aggregator
pip install -r requirements.txt
\`\`\`

### Usage
\`\`\`bash
python src/main.py
python src/main.py --coin bitcoin
python src/main.py --export json
python src/dashboard.py
\`\`\`
